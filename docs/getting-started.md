### GitHub Learning Path

GitHub is where you'll store your code, collaborate with teammates, and track your progress. Start with these beginner-friendly resources:

### VS Code Learning Path

VS Code is your primary code editor. It's powerful yet beginner-friendly with excellent extensions.

#### Official VS Code Documentation

- **[Getting Started](https://code.visualstudio.com/docs/getstarted/getting-started)** - Installation and first steps
- **[User Interface](https://code.visualstudio.com/docs/getstarted/userinterface)** - Understanding the editor layout
- **[Settings](https://code.visualstudio.com/docs/getstarted/settings)** - Customizing your environment
- **[Extensions](https://code.visualstudio.com/docs/editor/extension-marketplace)** - Adding functionality

#### VS Code Tutorials

- **[Intro Videos](https://code.visualstudio.com/docs/getstarted/introvideos)** - Short video introductions
- **[Tips and Tricks](https://code.visualstudio.com/docs/getstarted/tips-and-tricks)** - Productivity features

#### Essential Extensions for Your Work

Install these extensions to enhance your development experience:

- **GitLens** - Enhanced Git capabilities
- **Prettier** - Code formatting
- **ESLint** - JavaScript/TypeScript linting
- **Live Server** - Local development server
- **GitHub Pull Requests and Issues** - Direct GitHub integration
